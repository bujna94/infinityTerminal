Scroll left/right to add columns
∞ Infinity Terminal

Keyboard Shortcuts

  • Cmd/Ctrl + Shift + H — Home (scroll to first two columns)
  • Cmd/Ctrl + Shift + ← — Add column to the left
  • Cmd/Ctrl + Shift + → — Add column to the right
  • Cmd/Ctrl + / — Show this shortcuts popup
  • Cmd/Ctrl + Shift + M — Toggle minimap
  • Option + Command + ←/→ (macOS) — Step a column; hold to smooth-scroll
Press Esc or Cmd/Ctrl + / to close
